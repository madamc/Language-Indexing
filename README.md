Language-Indexing

# Setup
## Install locally:
```
pip install .
```

# Run
```
language-indexing
```

# Develop
## Use a virtual environment:
```
python -m venv env
```
## Activate virtual environment:
### Mac/Unix/Linux:
```
source env/bin/activate
```
### Windows:
```
env\Scripts\activate.bat
```
## Install in virtual environment
```
pip install .
```
